# graph_based_image_segmentation
graph_based_image_segmentation


implement of Efficient graph-based image segmentation [1]_.

## Build in linux
 mkidr build
 cd build
 cmake ..
 make

## Example

 ./bin/run_seg /path/to/image.jpg
